Ni Order and Chat for WooCommerce provide the option to chat and order directly from the shop owner.
 
== Description ==

<p>
Ni Order and Chat on WhatsApp for WooCommerce provide an option where customers can directly send the product related query directly to the store owner via whatsapp.Â The plugin provides the whatsapp button on each product detail page just below the add to cart button when customer or web visitor click on the whatsapp button then it sends the product name, product url and product sku directly on the whatsapp.
</p>
<p>
Whatsapp floating button can display at anycorner of the webpage, just change the setting to change the position ofwhatsapp floating button. Whatsapp floating provides an option to directly chatwith the shop owner on whatsapp.
</p>
<p>
Dashboard show the today sales indicator and top product sold by quantity and by sales value.
</p>


<h4>WhatsApp button setting for product detail page.</h4>
<ul>
	<li>Open whatsapp chatson the new tab if you check the then whatsapp windows open in the other tab.</li>
	<li>Enter whatsappnumber with country code, for example india country code 91 and then 10 digitmobile number 91XXXXXXXXXX.</li>	
	<li>Change the whatsappbutton text on the product detail page.</li>	
	<li>Message to the shopowner, include the product name, product url, product sku and product categoryin message. Also you can beautify the message with bold italic and strike.</li>	
</ul>


<h4>WhatsApp floating button or chat button setting.</h4>
<ul>
	<li>Open the whatsapp floatbutton on the new tab, if you check the then whatsapp windows open in the other tab.</li>
	<li>Whatsapp floating button position,Left bottom, right bottom, left top and right top.</li>	
	<li>Change the floating orchat button background color and fore color.</li>	
	<li>Enter the message for the shop owner.</li>	
</ul>

== Installation ==

1. Use the WordPress plugin installer or upload files to WordPress plugin directory.
2. Activate "Ni Order and Chat on WhatsApp for WooCommerce" plugin from admin panel.
3. Find menu "Ni WhatsApp".
